U Substitution

What is it?

U-sub is a method of integration that is useful when the function is either difficult to integrate, or cannot be integrated. It is a basic and frequently used method in calculus.

How it Works (in Words)

When you are given an integral that is difficult to integrate, set part of the function equal to U, and then find its derivative, du. You should now be left with a simple integration function in terms of U. Find the integral, then plug in the part of the function you set equal to U.
